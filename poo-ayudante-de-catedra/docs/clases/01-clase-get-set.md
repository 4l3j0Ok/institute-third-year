# Clase Get y Set + tips

## Esquema del documento

Se puede utilizar el esquema del documento para navegar rápidamente por las secciones. Para mostrar el esquema, ve a "Ver" > "Otras Ventanas" > "Esquema del documento" o presiona `Ctrl + Alt + T`.

El esquema muestra la estructura del código, incluyendo clases, métodos y propiedades. Puedes hacer clic en cualquier elemento para saltar directamente a esa sección del código.

En Windows Forms es especialmente útil para navegar entre los diferentes controles y eventos que hayas definido en tu formulario.

## Layouts disponibles en Windows Forms

Uso de Panel para agrupar controles relacionados y aplicar un diseño específico a ese grupo.

Uso de tablas con TableLayoutPanel para organizar controles de manera flexible. Permite definir filas y columnas, y ajustar el tamaño de los controles automáticamente.

Uso de FlowLayoutPanel para organizar controles en una dirección (horizontal o vertical) y que se ajusten automáticamente al tamaño del contenedor.

Uso de SplitContainer para dividir el formulario en secciones ajustables por el usuario.

Uso de Dock y Anchor para controlar cómo se ajustan los controles al redimensionar el formulario.

Dock permite que un control se adhiera a un borde del formulario.

Anchor permite que un control mantenga su posición relativa a los bordes del formulario.
