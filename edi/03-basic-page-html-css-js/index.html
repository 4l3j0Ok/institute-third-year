Sobre mí

Soy Alejo Sarmiento, SRE y Desarrollador de software.

Trabajo actualmente como SRE en Leafnoise hace más de 5 años. Primero como desarrollador backend y luego como SRE, donde me especialicé en la gestión de infraestructura en la nube y la automatización de procesos.

En cuanto a desarrollo backend, tengo amplia experiencia en Python con FastAPI, aunque también he trabajado con .NET y el ecosistema de JavaScript.

En cuanto a desarrollo frontend, tengo experiencia trabajando con Astro, React y Blazor, habiendo desarrollado varios proyectos personales y colaborativos utilizando estas tecnologías, como mi página web personal alejoide.com, mi página de reparación de computadoras alejoide.tech y otros tantos proyectos personales.

En cuanto a mi especialidad como SRE, me he enfocado en la gestión de infraestructura en la nube y on-premise, utlizando tecnologías como Docker y Kubernetes, implementando soluciones enfocadas a la integración continua y la entrega continua (CI/CD) para optimizar los procesos de desarrollo y despliegue, y asegurando la alta disponibilidad y el rendimiento de los sistemas a través de prácticas de monitoreo, escalabilidad y gestión de incidentes.

Mis tecnologías favoritas

  1. Python: FastAPI, Flask, PySide / PyQT
  2. JavaScript: Astro, React, Next.js, TypeScript, Tailwind CSS
  3. C#: .NET, ASP.NET, Blazor, Windows Forms
  4. Contenedores: Docker, Podman, Kubernetes, ArgoCD, Helm
